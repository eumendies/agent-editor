Agent V2 Orchestration Demo

同一个任务，三种不同的 agent 编排策略。这个页面不是普通编辑器，而是用来演示 ReAct、Planning 和 Supervisor 在任务拆解、执行路径与结果收口上的差异。

Single Agent: 即时决策与工具调用
Planning: 先拆计划，再逐步执行
Supervisor: 串行调度异构 worker 并最终汇总
Reflexion: actor 与 critic 交替迭代直到通过
